# cat-dog-id-mobile

## Dependencies

- Node.js (LTS)
  - https://nodejs.org

## Install & Run Project

- Clone git cat-dog-id-mobile repository
  - `git clone https://github.com/gsweissheimer/cat-dog-id-mobile.git`
- Move to the app repo
  - `cd cat-dog-id-mobile`
- Install dependecies
  - `npm install`
- Run project
  - `npx expo start`
